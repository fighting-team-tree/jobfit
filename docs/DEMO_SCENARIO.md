## 🎬 데모 시나리오

### 시나리오 1: 프로필 분석 (2분)

1. **랜딩 페이지** 접속
   - "이력서 진단" 카드 클릭
   
2. **프로필 페이지** (/profile)
   - 위 샘플 이력서 복사 → 붙여넣기
   - "분석하기" 버튼 클릭
   - ⏳ (NVIDIA NIM 분석 대기)
   
3. **결과 확인**
   - 추출된 기술 스택 태그 확인
   - 경력 요약 확인
   - "다음: 채용공고 분석" 클릭

---

### 시나리오 2: 갭 분석 (2분)

1. **대시보드 페이지** (/dashboard)
   - 위 샘플 JD 복사 → 붙여넣기
   - "갭 분석하기" 클릭
   - ⏳ (NVIDIA NIM 분석 대기)

2. **결과 확인**
   - 매칭 점수 확인 (예: 65%)
   - ✅ 보유 역량: Python, PostgreSQL, Docker...
   - ⚠️ 부족 역량: Java/Kotlin, Spring Boot, Kubernetes, Kafka...

3. **학습 로드맵 미리보기**
   - 4주 학습 계획 카드 확인
   - "모의 면접 시작" 클릭

---

### 시나리오 3: AI 모의 면접 (3분)

1. **면접 페이지** (/interview)
   - "면접 시작하기" 클릭
   - ⏳ (세션 생성 + 첫 질문 생성)

2. **첫 번째 질문**
   - AI 면접관 질문 확인
   - 🎙️ 마이크 버튼 클릭 → 음성 답변
   - 또는 텍스트 직접 입력
   - "답변 제출" 클릭

3. **2-3개 추가 질문 응답**
   - 꼬리물기 질문 체험
   - 질문 카운터 확인 (2/5, 3/5...)

4. **면접 종료**
   - 5개 질문 완료 또는 "종료" 버튼
   - 피드백 페이지로 이동
